Movie Database: Gerenciamento de Filmes com Django

O Movie Database é uma aplicação web desenvolvida com Django e Django REST Framework (DRF) que permite gerenciar um catálogo de filmes de forma eficiente. Com uma interface dinâmica construída com Django Template, os usuários podem visualizar e cadastrar filmes, enquanto a API REST facilita a integração e manipulação dos dados.

📌 Funcionalidades

🔍 Consulta de filmes: Pesquise e visualize detalhes de filmes cadastrados. Consulte filmes

📄 CRUD completo: Criação, leitura, atualização e exclusão de registros.Cadastre filmes

🌍 Integração com API externa: Busca automática de informações sobre filmes. API consumida

🗄 Banco de dados PostgreSQL: Armazena os dados de forma segura e escalável.

🐳 Ambiente Dockerizado: Facilita a configuração e a implantação do projeto.
